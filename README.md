## Overview

Riona automates social media workflows with AI-generated captions and comments, scheduled posting, and engagement automation. The landing page introduces the product, highlights core capabilities, and directs visitors to onboarding.

## What Riona Does

- Automates Instagram and X interactions (post, like, comment)
- Generates captions and replies with Gemini + schema-guided output
- Trains from YouTube, audio, documents, and websites
- Applies safety controls (cooldowns, limits, retries)

## Tech Stack

- Next.js
- React
- Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
